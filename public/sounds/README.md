# Sound Files

Please add the following sound files to this directory:

1. **countryside.mp3** (or countryside.ogg) - Background countryside/ambient sound
   - Should be a looping ambient sound that matches the 3D scene theme
   - Recommended: Nature sounds, countryside ambience, peaceful background music

2. **click.mp3** (or click.ogg) - Button click sound effect
   - Short, subtle click sound for navigation buttons
   - Recommended: Soft click, button press, or gentle tap sound

## File Formats Supported
- MP3 (.mp3)
- OGG (.ogg)

The audio player will automatically try both formats for better browser compatibility.

## Getting Sound Files
You can find free sound effects at:
- https://freesound.org
- https://pixabay.com/music/
- https://mixkit.co/free-sound-effects/

For countryside sounds, search for: "countryside", "nature ambience", "rural sounds", "peaceful nature"

For click sounds, search for: "button click", "soft click", "tap sound", "ui click"
